# 📅 Reservas de Salas – API REST

Sistema backend para gerenciamento de reservas de salas, usuários e controle de conflitos de agendamento, desenvolvido com Spring Boot.

---

## 🚀 Tecnologias utilizadas

- Java 21
- Spring Boot 3
- Spring Web
- Spring Data JPA
- Hibernate
- PostgreSQL (produção)
- H2 (testes)
- Maven
- JUnit 5
- Mockito

---

## 🧠 Funcionalidades

### 👤 Usuários
- Criar usuário
- Listar usuários
- Buscar usuário por ID
- Atualizar usuário
- Remover usuário

### 🏢 Salas
- Criar sala
- Listar salas
- Buscar sala por ID
- Atualizar sala
- Remover sala

### 📌 Reservas
- Criar reserva
- Listar reservas
- Buscar reserva por ID
- Cancelar reserva
- Validação de conflito de horários
- Controle de status (ATIVA / CANCELADA)
